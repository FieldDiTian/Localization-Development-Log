**激光雷达 LiDAR**
luminar_front / luminar_left / luminar_right 是 **Luminar/Iris 激光雷达 LiDAR**，三颗不同位置的 lidar
在这个仓库里对应的是三颗 lidar：
- luminar_front：前向激光雷达，sensor_id 20
- luminar_right：右侧激光雷达，sensor_id 21
- luminar_left：左侧激光雷达，sensor_id 22

radar_front / radar_rear 是 **毫米波雷达 Radar**，前后两颗不同雷达

**离线建图：GLIM**
用 3 个 Luminar Iris LiDAR、Point One Atlas INS/RTK、IMU，跑 GPU 加速 LiDAR-inertial SLAM，生成全局一致的 3D 地图、轨迹和 T_world_utm.txt 

**在线定位：gicp_localization**  
    加载 GLIM 产出的 PCD 地图，用实时 LiDAR scan-to-map GICP 定位；IMU 在两帧 LiDAR 之间做 dead-reckoning，并以约 IMU 频率发布 odom

传感器/rosbag/pcap
   ↓
GLIM 建图
   ↓
人工检查/导出 PCD 地图
   ↓
gicp_localization 在线定位
   ↓
车辆/下游系统使用的 pose / odom / path

**输入**

主要输入是 AV-24 车上的传感器数据：

- LiDAR 点云：/luminar_front/points，以及可拼接的 /luminar_left/points、/luminar_right/points
- IMU：/gps_p1/imu
- Atlas INS / RTK odom：/gps_p1/filtered_odom
- 车辆外参：av24.urdf
- 配置文件：GLIM 的 JSON config、gicp_localization/cfg/localization.yaml
- 离线模式还可以输入 rosbag、mcap，或 Luminar raw .pcap
